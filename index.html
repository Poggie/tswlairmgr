How to use

Start out by selecting your lair from the dropdown menu in the upper right corner.

Click the + and - buttons to select how many of each fragment you have.

  • The first number below each fragment is the count you already have in your inventory (e.g. brought with you or looted from monsters while doing the lair missions), and it is also the number you will modify by clicking the buttons.
  • The second number in braces tells you how many you will have in total after collecting the mission reward fragments from your team according to the pick table.

Note: Changing the number of fragments you have will instantly recalculate and update the pick table accordingly. It will not respect any previously assigned and received fragments and completely mix things up.

The fragment(s) with the lowest count will be marked in yellow for each boss so you can at a glance see which fragment drops to look out for when farming more or searching the marketplace.

Note: This will only consider the number of fragments you currently have, as the pick table will be adjusted accordingly if you add another fragment to the boss fragment table.

See at a glance how many boss summons you will get for each boss (this counts in fragments from the pick table) and also how many distinct fragments you will need to get another full set.

Note: The number of missing fragments will not always resemble the number of marked fragments as the markings will only consider which fragments you already have, whereas the calculations here take into account the fragments you will get as mission rewards.

You can either add your team members before or after the run.

  • Adding them before the run will allow you to predict the number of boss summons calculating in the fragments you will get from your team members.
  • Adding them afterwards can make the markings on the boss fragment table a little less confusing.

Tip: You don't have to click the Add button after entering each name; simply pressing Return while still having the text box selected will also add the name to the list. This makes it very easy and fast to enter multiple names in sequence.

Untick the respective checkbox(es) next to a name – one for each mission/boss, same order as in the boss fragment table at the top – to tell the app that this team member can not turn in those missions and needs to be skipped at fragment assignment.

Note: This will instantly update the pick table.

See at a glance which team member should pick which fragments as their mission rewards.

Click the fragments to check them off (reduce opacity) for a better overview about who has traded you which fragments already and who still has to.

Note: The order of fragments/missions/bosses is the same as in the boss fragment table at the top.

Warning: Any changes you make in the boss fragment and participants tables will reset the checked/unchecked states here.

Click the download button to download a TSW chat script which will output the pick table, sorted by mission/boss, in the currently selected chat channel.

Warning: This will post many lines of text at once, so make sure you have a "private" channel selected first before firing it, such as Group, Raid or Say chat.
Please do not spam global channels.

Save the file in the Scripts subfolder located in your installation folder of TSW.
If there is no folder with that name inside your installation folder, you will have to create it manually.

If you are asked if you want to overwrite an already existing file, double check the file name in the Save file dialog:

  • If the name is still tswlairmgr (without any file extension), you can safely choose to overwrite as you will likely not want to keep multiple TSW lair manager scripts at the same time to avoid confusion.
  • If you find a different name in the file name field, you must have accidentally marked a file in the Save file dialog. The easiest way to resolve this is to simply cancel at that point and clicking the download button again.

In the TSW client, click the text in square brackets in the lower left corner of your chat window to bring up a menu to select into which chat channel you want to post.

  • Select Group if all your team's members are in a 5-person group.
  • Select Raid if all your team's members are in a 10-person raid (double group).
  • Select Say if you have more team members than what you can fit into one raid. Make sure all team members are standing close to you before you fire the script as people will only receive Say messages within a certain radius around you.

Then enter /tswlairmgr (assuming you haven't changed the file name) as the chat message and press Return.

Tip: You can also just type the first few characters (e.g. /tswl) and then press Tab to either instantly complete the script name or display a selection menu containing all scripts beginning with those characters in which you can then use ↑ and ↓ to select an item and press Return to accept it.

The pick table will now be posted, sorted by mission/boss, in chat.
